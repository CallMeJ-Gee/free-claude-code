Provider Setup Guides

🟢
NVIDIA NIM
High-quality hosted models
  1. Get an API key at build.nvidia.com/settings/api-keys
  2. Go to Providers and add NVIDIA NIM
  3. Enter your API key and save
  4. Test the connection
Popular models: nvidia_nim/z-ai/glm4.7, nvidia_nim/z-ai/glm5, nvidia_nim/moonshotai/kimi-k2.5
🟣
OpenRouter
Many models including free options
  1. Get an API key at openrouter.ai/keys
  2. Go to Providers and add OpenRouter
  3. Enter your API key and save
  4. Test the connection
Free models: open_router/stepfun/step-3.5-flash:free, open_router/deepseek/deepseek-r1-0528:free
🔵
DeepSeek
Affordable high-quality models
  1. Get an API key at platform.deepseek.com/api_keys
  2. Go to Providers and add DeepSeek
  3. Enter your API key and save
  4. Test the connection
Popular models: deepseek/deepseek-chat, deepseek/deepseek-coder
💻
LM Studio
Local models with GUI
  1. Download and install LM Studio
  2. Load a model in LM Studio
  3. Start the local server (usually on port 1234)
  4. Go to Providers and add LM Studio
  5. Enter the server URL (default: http://localhost:1234/v1)
  6. Test the connection
🦙
Ollama
Local models with CLI
  1. Download and install Ollama
  2. Pull a model: ollama pull llama3.1
  3. Start the server: ollama serve
  4. Go to Providers and add Ollama
  5. Enter the server URL (default: http://localhost:11434)
  6. Test the connection
